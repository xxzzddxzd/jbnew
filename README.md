# jbnew
 适用于iOS 14的触摸模拟+LUA+图像识别（opencv2）

# 一些说明
1、opencv2的库太大，不让传。自行编译arm64e的版本，放在f目录下即可。
2、插件编译好以后在越狱机老位置
3、脚本目录放在/script/xxx/， xxx是目录，来区分各个脚本的，目录里是图片文件，脚本必须名为1234.lua
4、脚本范例在simpleLua里。
5、重启backboardd进程后，音量键调起对话框，选择要执行的脚本即可。再次按下为停止运行。
